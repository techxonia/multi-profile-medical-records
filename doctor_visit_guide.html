Doctor Visit Setup Guide

Q. How to add a doctor visit?

A. Follow the step below:

  1.   1. Open Multi-Profile Medical Records Application from home screen.
  2.   2. You will redirect to the “Profile” dashboard screen.
  3.   3. Press the "View Records" button to see all of your or your family's medical records.
  4.   4. Press "Doctor Visit" to go to the next page where you can see all the doctor visit lists.
  5.   5. At the bottom of the application you can see the plus icon at the right side. If you press that plus icon then you will get two options either to import from excel or to enter manually.
  6.   6. To import from Excel, click "Import from Excel", which will take you to the next page where you can easily import from your device.
  7.   7. For manual entry, click "Manual Entry" , which will take you to the next page where you can easily enter the details about the doctor visit manually.

Q. How to copy Excel Format File? (Laptop Work)

A. Follow the step below: (Video tutorial 1:30 -1:50)

  •   1. Open the Doctor Visit Excel Format Sheet "Doctor Visit Excel File". (Recommend to use your desktop or laptop)
  •   2. On the top left section you will see File tab.
  •   3. Click on File tab, go to Make a Copy section.
  •     a. Make sure you're signed in to your Google Account. If not, you won't see the "Make a Copy" section.
  •   4. Rename the file (e.g Rename "Your_Name Doctor Visit Excel Format"). And you can edit the Excel file that you have created.

Q. How to edit the Excel format file you create? (Laptop Work)

A. Follow the step below: (Video tutorial 2:00 -3:00)

  •   1. Open the file ("Your_Name Doctor Visit Excel Format" ) that you have copied and rename it in the prevoius step.
  •   2. Now you are free to add as much data as you want.
  •     a. Don't add any extra column or remove column from given excel sheet.
  •     b. Don't change the cloumn format & validation from excel sheet.
  •     c. Don't alter any cloumn position.
  •     d. Double click on Visiting Date Cloumn to pick the date.
  •       i. Keep the first row of the Excel sheet as it is. And enter the data from the second row.
  •       ii. Each row first cloumn must be filled in with the title, otherwise the data in that row will not be inserted in the application.
  •       iii. Please make sure you enter the data correctly in each respective column. Otherwise, your data will not be inserted or will not match in the application.
  •   3. Enter the number of row as you wish to add your doctor visit data and click the "Add" button.

Q. How to download the Excel format file you create? (Laptop Work)

A. Follow the step below: (Video tutorial 3:07 -3:26)

  •   1. Open the file ("Your_Name Doctor Visit Excel Format" ) excel sheet file.
  •   2. On the top left section you will see File tab.
  •   3. Click on File tab, go to Download Section.
  •   4. You will see lots of option there, just click on the Microsoft Excel (.xlxs).
  •   5. Go to "Downloads folder" to view the downloaded file.

Q. How to transfer downloaded excel sheets from desktop/laptop to smartphone? (Laptop Work)

A. You can share it via bluetooth, email, social media or any other medium.

Q. How to import excel file? (Smartphone Work)

A. Follow the step below: (Video tutorial 3:40 -3:59)

  •   1. Open Multi-Profile Medical Records Application from home screen.
  •   2. You will redirect to the “Profile” dashboard screen.
  •   3. Press the "View Records" button to see all of your or your family's medical records.
  •   4. Press "Doctor Visit" to go to the next page where you can see all the doctor visit lists.
  •   5. At the bottom of the application you can see the plus icon at the right side. If you press that plus icon then you will get two options either to import from excel or to enter manually.
  •   6. To import from Excel, click "Import from Excel", which will take you to the import excel data page.
  •   7. Click on Import Excel file button, it will redirect you to the default android file manager.
  •     a. Make sure you have saved the Excel file that you edited and downloaded in the previous steps.
  •     b. Find the file where you have saved or downloaded.
  •     c. Click on the Excel file you want to import in the doctor visit.
  •   8. Clicking on the Excel file will redirect you to the doctor visit page. And you'll see the data you've imported from Excel.
